# Continental Edison Smart TV - CELD55SQLDV24B6

**IR remote file for Flipper Zero**

| Item | Value |
|------|-------|
| Protocol | NECext |
| Address | 00 7F 00 00 |
| Model | CELD55SQLDV24B6 |

## ✅ Working buttons

Power, Vol_up/dn, Mute, Ch_up, Up, Down, Left, Right, OK, Input, Satellite, USB, Settings, App_Menu, Account, Record, Netflix, YouTube, Prime_Video, Disney_Plus, Pair_Remote

## 📥 Usage

1. Download `ContinentalEdison_CELD55SQLDV24B6.ir`
2. Copy to `infrared` folder on Flipper Zero
